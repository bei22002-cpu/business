# Business Website

A professional single-page business website built with HTML and CSS.

## Pages / Sections

- **Hero** – Headline, tagline, and call-to-action buttons
- **Services** – Six service cards (Strategy, Digital Transformation, Marketing, Operations, Risk, Analytics)
- **About** – Company overview with key statistics
- **Testimonials** – Three client reviews
- **Contact** – Contact details and an enquiry form

## Getting Started

Open `index.html` in any modern web browser — no build step required.

```bash
open index.html   # macOS
start index.html  # Windows
xdg-open index.html  # Linux
```

## Files

| File | Description |
|------|-------------|
| `index.html` | Main page markup |
| `styles.css` | All styles (responsive) |
